<h3>课程目标：</h3>
<p>本课程的目标是学会使用<a title="查看与 Django 相关的文章" target="_blank">Django</a>框架的基础使用。与其他Django基础课程不同的是，这个课程每个知识点都讲解非常仔细，知其然也知其所以然，清楚的知道自己写的每一行代码代表的是什么意思。</p>
